Git Terms Assignment

Version Control Software

Version Control Software is a software that keeps track of changes to source code. It is used a lot for code that is being worked on by many people in order to track different changes several people have made to the code over time.

add

Add is a command in git that is used to stage, or add, files to a repo. You can use it to add a single file or all files.

Source for definition

commit

Commit is a command in git that is used to save changes done to a file locally.

Source for definition

push

Push is a command in git that allows you to push your local changes to a file that were committed to the remote branch.

Source for definition

pull

Pull is a command in git that allows you to get, or fetch, all the changes done to a file on a remote repository, and also merges the changes done remotely on your local branch.

Source for definition

clone

Clone is a command in git that is used to download the most recent version of a remote project to your local computer.
